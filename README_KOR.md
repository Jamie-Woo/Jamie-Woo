### 소개
-  데이터 분석과 국제개발 분야에 관심 있는 대학생입니다.
-  입지선정 분석과 예측 모델링에 대한 학술 프로젝트 경험이 있습니다.
-  KOICA-UNV 프로그램을 통해 엘살바도르 세계식량계획(WFP)에서 현장 봉사활동을 수행하였습니다.
